## 模板使用说明

### 状态流转

```
TODO → DOING → REVIEW → DONE
              ↓
           BLOCKED
```

### 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 任务编号 | DEV-XXX 格式 | DEV-001, DEV-002 |
| 状态 | 当前任务状态 | TODO: 待开始<br>DOING: 进行中<br>REVIEW: 审核中<br>DONE: 已完成<br>BLOCKED: 被阻塞 |
| 优先级 | 任务优先级 | 高: 阻塞性任务<br>中: 正常任务<br>低: 优化性任务 |
| 验收标准 | 可测试的完成条件 | "用户可以登录" 而非 "实现登录功能" |

### 与 OpenClaw/Claude Code 对接

1. **OpenClaw 下发任务**：使用此模板创建任务文件
2. **Claude Code 执行**：按照执行模板记录到"执行记录"部分
3. **任务完成**：更新状态为 DONE，填写"完成说明"
