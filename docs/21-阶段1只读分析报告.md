## 3. OpenViking 核心能力分析

### 3.1 AGFS 文件系统（文件系统范式）

**AGFS（AI Agent File System）**：基于文件系统范式统一管理智能体所需的记忆、资源和技能。

**数据目录结构**：
```
/root/.openviking/data/viking/
├── user/                          # 用户数据
│   ├── memories/                   # 记忆数据
│   │   ├── entities/              # 实体
│   │   ├── events/                # 事件
│   │   └── preferences/           # 偏好
├── session/                        # 会话
├── resources/                     # 资源
└── agent/                         # Agent
    ├── skills/                    # 技能
    └── instructions/              # 指令
```

**三级内容访问**：
- **L0（abstract）**：摘要级别，快速了解要点
- **L1（overview）**：概览级别，了解详细内容
- **L2（read）**：完整级别，获取完整内容

每个目录都有：
- `.abstract.md`（L0 摘要）
- `.overview.md`（L1 概览）
- 实际内容文件（L2 完整内容）

## 4. VikingBot 8 个 Agent 工具分析

### 4.1 openviking_read（资源读取）

**功能**：读取 OpenViking 资源，支持三级访问

**参数**：
- `uri`：OpenViking 文件的 URI（如 viking://resources/path/123.md）
- `level`：读取级别（abstract/overview/read），默认 abstract

**能力**：
- L0（abstract）：快速获取摘要
- L1（overview）：获取概览
- L2（read）：获取完整内容

---

### 4.2 openviking_list（资源列表）

**功能**：列出 OpenViking 路径下的资源

**参数**：
- `uri`：父级 Viking URI（如 viking://resources/bot_test/dutao/）
- `recursive`：是否递归列表，默认 false

**能力**：
- 递归遍历目录结构
- 列出所有资源

---

### 4.3 openviking_search（语义搜索）

**功能**：使用查询搜索 OpenViking 资源

**参数**：
- `query`：搜索查询
- `target_uri`：可选的搜索范围 URI，默认整个范围

**能力**：
- 语义相似度检索
- 精准定位相关资源

---

### 4.4 openviking_add_resource（添加资源）

**功能**：添加本地文件为 OpenViking 资源

**参数**：
- `local_path`：本地文件路径
- `description`：资源描述
- `target_path`：Viking 中的目标路径（如 /bot_test/dutao/docs/）
- `wait`：是否等待处理完成，默认 false

**能力**：
- 本地文件导入
- 自动向量化
- 自动生成摘要和概览

---

### 4.5 openviking_grep（正则搜索）

**功能**：使用正则表达式搜索 OpenViking 资源

**参数**：
- `uri`：搜索范围 URI（如 viking://resources/path/）
- `pattern`：正则表达式模式
- `case_insensitive`：是否忽略大小写，默认 false

**能力**：
- 精确文本搜索
- 正则表达式匹配
- 大小写敏感/不敏感

---

### 4.6 openviking_glob（Glob 查找）

**功能**：使用 Glob 模式查找 OpenViking 资源

**参数**：
- `pattern`：Glob 模式（如 **/*.md, *.py, src/**/*.js）
- `uri`：搜索范围 URI，默认空字符串

**能力**：
- 通配符模式匹配
- 递归查找
- 支持常见 Glob 模式

---

### 4.7 user_memory_search（用户记忆搜索）

**功能**：搜索用户记忆

**参数**：
- `query`：搜索查询

**能力**：
- 语义搜索用户记忆
- 快速检索历史对话
- 个性化结果

---

### 4.8 openviking_memory_commit（记忆提交）

**功能**：提交消息到 OpenViking 会话

**参数**：
- `messages`：消息列表，每个消息包含 role、content、可选的 tools_used

**能力**：
- 持久化对话历史
- 自动压缩和摘要
- 提取长期记忆
